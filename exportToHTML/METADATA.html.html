<span class="s0">```python3</span>

<span class="s0">questionary.text(&quot;What's your first name&quot;).ask()</span>

    <span class="s0">&quot;What do you want to do?&quot;,</span>

<span class="s0">).ask()</span>

<span class="s0">questionary.rawselect(</span>
    <span class="s0">&quot;What do you want to do?&quot;,</span>
    <span class="s0">choices=[&quot;Order a pizza&quot;, &quot;Make a reservation&quot;, &quot;Ask for opening hours&quot;],</span>
<span class="s0">).ask()</span>

<span class="s0">questionary.checkbox(</span>
    <span class="s0">&quot;Select toppings&quot;, choices=[&quot;foo&quot;, &quot;bar&quot;, &quot;bazz&quot;]</span>

<span class="s0">## Features</span>

<span class="s0">Questionary supports the following input prompts:</span>
 
 <span class="s0">* [Text](https://questionary.readthedocs.io/en/stable/pages/types.html#text)</span>
 <span class="s0">* [Password](https://questionary.readthedocs.io/en/stable/pages/types.html#password)</span>
 <span class="s0">* [File Path](https://questionary.readthedocs.io/en/stable/pages/types.html#file-path)</span>
 <span class="s0">* [Confirmation](https://questionary.readthedocs.io/en/stable/pages/types.html#confirmation)</span>
 <span class="s0">* [Select](https://questionary.readthedocs.io/en/stable/pages/types.html#select)</span>
 <span class="s0">* [Raw select](https://questionary.readthedocs.io/en/stable/pages/types.html#raw-select)</span>
 <span class="s0">* [Checkbox](https://questionary.readthedocs.io/en/stable/pages/types.html#checkbox)</span>
 <span class="s0">* [Autocomplete](https://questionary.readthedocs.io/en/stable/pages/types.html#autocomplete)</span>

<span class="s0">There is also a helper to [print formatted text](https://questionary.readthedocs.io/en/stable/pages/types.html#printing-formatted-text)</span>
<span class="s0">for when you want to spice up your printed messages a bit.</span>

<span class="s0">## Usage</span>

<span class="s0">```python</span>
<span class="s0">import questionary</span>

<span class="s0">questionary.select(</span>
    <span class="s0">&quot;What do you want to do?&quot;,</span>
    <span class="s0">choices=[</span>
        <span class="s0">'Order a pizza',</span>
        <span class="s0">'Make a reservation',</span>
        <span class="s0">'Ask for opening hours'</span>
    <span class="s0">]).ask()  # returns value of selection</span>
<span class="s0">```</span>

<span class="s0">That's all it takes to create a prompt! Have a [look at the documentation](https://questionary.readthedocs.io/)</span>
<span class="s0">for some more examples.</span>
