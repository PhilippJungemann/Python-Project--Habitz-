<pre><span class="s0">&quot;&quot;&quot;distutils.errors 
 
Provides exceptions used by the Distutils modules.  Note that Distutils 
modules may raise standard exceptions; in particular, SystemExit is 
usually raised for errors that are obviously the end-user's fault 
(eg. bad command-line arguments). 
 
This module is safe to use in &quot;from ... import *&quot; mode; it only exports 
symbols whose names start with &quot;Distutils&quot; and end with &quot;Error&quot;.&quot;&quot;&quot;</span>

    <span class="s2">pass</span>

    <span class="s0">&quot;&quot;&quot;Unable to load an expected module, or to find an expected class 
    within some module (in particular, command modules and classes).&quot;&quot;&quot;</span>

    <span class="s2">pass</span>


<span class="s2">class </span><span class="s1">DistutilsClassError(DistutilsError):</span>
    <span class="s0">&quot;&quot;&quot;Some command class (or possibly distribution class, if anyone

    <span class="s2">pass</span>
